# Ecommerce API

## Descrição
Este projeto é uma API para um sistema de e-commerce, construído com Spring Boot. Ele fornece endpoints para realizar operações CRUD (Create, Read, Update, Delete) em produtos.

## Tecnologias Utilizadas
- Spring Boot
- Spring Data JPA
- Swagger
- Maven
